# My Bookmarks

View and manage pages you have bookmarked in the AI/Spec-Driven Textbook for Physical AI & Humanoid Robotics.

## How to Use Bookmarks

1. **Save a page**: Click the "Bookmark" button at the top of any chapter page
2. **View bookmarks**: Return to this page to see all your saved pages
3. **Navigate**: Click any bookmarked page title to return to that content
4. **Manage**: Remove bookmarks you no longer need using the "Remove" button

Your bookmarks are stored locally in your browser and will persist between sessions.

> Note: This page is designed to work with the bookmarking functionality built into the textbook interface. If you're seeing this page directly, the interactive bookmark features may not be working properly.
